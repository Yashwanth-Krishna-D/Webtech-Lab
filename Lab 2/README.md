# LAB-2 — Weather Dashboard

## Overview
Enhanced weather dashboard built with HTML, CSS and JavaScript. 
Features:
- Current weather lookup by city using OpenWeatherMap API.
- unit toggle (°C/°F), last-searched city saved in localStorage.

## Setup
1. Get an API key from OpenWeatherMap: https://openweathermap.org/api
2. Open `script.js` and replace `YOUR_API_KEY_HERE` with your key.
3. Open `index.html` in a browser.

## Files
- `index.html` — markup and UI
- `style.css` — styles
- `script.js` — JS logic and fetch
